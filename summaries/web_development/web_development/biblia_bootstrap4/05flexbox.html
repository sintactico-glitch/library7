Trabajando con d-flex

Todo el sistema de columnas de Bootstrap está basado en flexbox. Las clases .row tienen display: flex, lo que permite que las .col se coloquen una al lado de la otra

Clases para crear un flex container:

  • .d-flex
  • .d-sm-flex
  • .d-md-flex
  • .d-lg-flex
  • .d-xl-flex

Si queremos sacar el entorno flexible asignamos la clase .d-block (con sus variables respectivas dependiendo de los contextos)

Cambiando la orientación con flexbox

Para cambiar la orientación le agregamos las siguientes clases al flex container:

  • .flex-row
  • .flex-column
  • .flex-row-reverse
  • .flex-column-reverse
  • .flex-md-row, etc
  • .flex-md-column, etc

Alineación en el eje principal con flexbox

Las siguientes clases se aplican al flex container:

  • .justify-content-start
  • .justify-content-center
  • .justify-content-end
  • .justify-content-between
  • .justify-content-around
  • .justify-content-evenly

Estas clases también se pueden utilizar de forma responsive, dependiendo de los distintos contextos que maneja Bootstrap. La estructura que se sigue es:

.justify-content-contexto-alineación

Por ejemplo: .justify-content-md-center
