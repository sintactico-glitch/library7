Understanding Client and Server

The client:

Is the device in front of you, it runs a web browser (software that runs on the client that displays webpages like Chrome, Firefox, etc)

The server:

It's where your public website is going to live

  • Hardware server: the physical computer that serves up web pages
  • Software server: program that executes the commands to send files out so the website can be seen (Apache)
