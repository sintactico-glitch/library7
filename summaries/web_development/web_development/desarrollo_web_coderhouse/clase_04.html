Estilos de texto

  • font-style
  • font-weight
  • font-size
  • font-family
  • text-align (mueve la posición en la que se encuentran los elementos de línea, entre ellos, textos, titulares, img )
  • text-transform (permite ignorar cómo escribimos el texto en HTML y después definir el estilo que aplica con mayúsculas y minúsculas)
  • line-height (interlineado, espacio que hay entre los renglones)
  • text-decoration (decorar el texto con líneas arriba, por encima o debajo del texto)
  • letter-spacing
  • word-spacing

Estilos de lista

  • list-style-type
  • list-style-image
  • list-style-position (indentación de las listas, valores inside/outside
  • list-style (abreviación de las propiedades anteriores)

Estilo de fondos

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-size

Unidades de medida

Absolutas

  • Px(pixels): es la unidad que usan las pantallas

Relativas

  • Rem: relativa a la configuración de tamaño de la raíz (etiqueta HTML). Es decir, si cambiamos el tamaño del font-size del HTML, se modifican todas las etiquetas que utilicen rem como medida
  • Porcentaje: teniendo en cuenta que 16px es el 100%
  • Viewport: dependen del tamaño de la pantalla del dispostivo que se use para ver la web. Van de 0 (0% del tamaño de pantalla) al 100 (100% del tamaño de pantalla)
