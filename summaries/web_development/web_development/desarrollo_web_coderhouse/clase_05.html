Propiedades de la caja

Todos los elementos de HTML son cajas

  • Elementos de línea: se ve uno debajo del otro (a y span)
  • Elementos de bloque: se ve uno debajo de otro (section, div, h1 al h6 )

Tip para diferenciarlos: en 99% de los casos, si estos elementos tienen que interactuar dentro de un texto (strong, b, span, a), siempre van a ser de línea. La idea es que interactuen sin que empujen el contenido hacia abajo y solamente ocupan el espacio necesario para entren sus hijos cómodamente

Propiedades en común

  • Content: es lo que ponemos, el espacio para el texto o la imagen
  • Padding: la primera capa después del contenido. Es la separación entre el borde y el contenido de la caja, es un espacio interior
  • Border: límite entre el elemento y el espacio externo
  • Margin: separación entre el borde y el afuera de la caja. Es un espacio exterior. Sirve para separar un contenido de otro. El margen separa los elementos pero no cambia el tamaño de la caja

Alto y ancho de los elementos

Width y Height controlan el ancho y altura de las cajas, respectivamente.

Ambas propiedades no admiten valores negativos y los valores en porcentaje se calculan a partir de la altura de su elemento padre.

Los elementos de línea no modifican su tamaño con las propiedades height y width. Solamente los elementos de bloque hacen esto.
